# Amash Ranasinghe — CSE Investment Intelligence Portal

**Live site:** https://amash-ranasinghe.github.io

A personal investment intelligence portal for the Colombo Stock Exchange, built by Amash Ranasinghe, AVP at Asia Securities (Pvt) Ltd.

## What's inside

- 📈 **Equity** — 15-year ASPI cycles chart, annual returns, sector performance
- 💰 **Fixed Income** — Money Market, Income & Dynamic Income Fund returns
- 🏆 **Stock Picks** — 17 Asia Securities BUY-rated stocks with target prices
- 📊 **CSE Returns** — All 289 CSE stocks ranked by 1-year return
- 💼 **Portfolio Tracker** — Enter your holdings, see your P&L vs ASPI
- 🌍 **Commodity Tracker** — Bloomberg data, 20 commodities
- 📋 **Macro Dashboard** — Full SL macro with Gulf War III analysis
- ⚖️ **Risk Profiler** — 5-question investor profile assessment
- 🔮 **Return Simulator** — Project growth across 6 investment scenarios
- 📞 **Contact** — Book a consultation directly
